Let $(X_0, Y_0)$ be the maximum point of $v$. Then

\begin{equation}
    \text{Max } v(x, t) = \text{Max } u(x, t) + 2.
\end{equation}

Thus, we have

\begin{align}
    U(x, t) &= v(x, t) - E x, \\
    \text{Max } u(x, t) &< \text{Max } U(x, t).
\end{align}

Finally, we conclude

\begin{equation}
    R = \text{Max } v(x, t) - \text{Max } U_t + 3.
\end{equation}
