\section*{Example 2}

\begin{align}
    U_x X + U &= 0, \\
    u'' + u &= 0.
\end{align}

The solution for \( u(x, y) \) is given by:
\[
u(x, y) = C_1(y) \cos x + C_2(y) \sin x
\]

\section*{Example 3}

\begin{align}
    U_{xy} &= 0, \\
    u_{xx} &= 0, \\
    u_x &= F(x).
\end{align}

The general solution is:
\[
u = \int F(x) \, dx + H(x) + G(y)
\]
